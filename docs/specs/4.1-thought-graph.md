# Feature 4.1: Thought graph / connection map

**Category:** Add as route to `dashboards/command-center`

## Done when

- Nodes = thoughts, edges = semantic similarity > 0.75
- Nodes colored by type, sized by connection count
- Hover highlights connections, click shows content in side panel
- Drag, zoom, pan all work
- Filter by type/topic/date range (same FilterBar from 1.2d)
- Works reasonably for up to ~500 thoughts

## What to build

### Data pipeline

1. Fetch thoughts with embeddings
2. Compute pairwise cosine similarity (client-side for <500 thoughts, or use RPC below for larger sets)
3. Create edges where similarity > 0.75 (configurable threshold)
4. Color nodes by `metadata->>'type'`, size by connection count

### Server-side RPC (for performance at scale)

```sql
create or replace function thought_graph(similarity_threshold float default 0.75)
returns table(source_id uuid, target_id uuid, similarity float) as $$
  select a.id, b.id, 1 - (a.embedding <=> b.embedding) as sim
  from thoughts a, thoughts b
  where a.id < b.id
    and 1 - (a.embedding <=> b.embedding) > similarity_threshold
  order by sim desc
  limit 500;
$$ language sql;
```
