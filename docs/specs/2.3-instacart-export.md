# Feature 2.3: Shopping list → Instacart export

**Category:** MCP tool addition to meal-planning MCP or `open-brain-mcp`

## Done when

- `export_shopping_list` MCP tool exists
- Aggregates ingredients across recipes for a given week's meal plan
- Groups by category (produce, protein, dairy, pantry, etc.)
- Returns formatted text block ready to paste into Instacart or any grocery app
- This test works:
  ```
  Export my shopping list for this week
  → Returns categorized, aggregated ingredient list
  ```

## What to build

### MCP tool

| Tool | Params | Description |
|------|--------|-------------|
| `export_shopping_list` | `week_start?` (defaults to current week) | Export aggregated shopping list for a week's meal plan. |

### Logic

1. Query `shopping_lists` for the given week
2. Join to `recipes` for ingredient details
3. Aggregate duplicates (2 recipes both need chicken → one line with combined quantity)
4. Group by category
5. Return formatted text

### Output format

```
Shopping List — Week of March 24

PRODUCE
- Broccoli, 4 cups
- Bell peppers, 2 cups
- Onion, 1 large

PROTEIN
- Chicken breast, 2 lbs

PANTRY
- Soy sauce, 6 tbsp
- Olive oil, 4 tbsp
```
