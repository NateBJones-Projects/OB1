# Feature 4.3: Capture source expansion

**Category:** `integrations/`

> This is a collection of modular integrations. Each can be a separate Claude Code session.

## 4.3a: Quick capture PWA

### Done when
- Single-page web app at `integrations/quick-capture/`
- One text field, one submit button
- Hits `ingest-thought` Edge Function directly
- Installable as PWA on phone (service worker + manifest)
- `README.md` + `metadata.json`

### What to build
Minimal PWA: `index.html`, `manifest.json`, `sw.js`. Text input, submit button, success/error feedback. Edge Function URL from config. No framework needed — vanilla HTML/JS.

---

## 4.3b: Voice memo capture

### Done when
- Audio file → transcription → `capture_thought`
- Works via Telegram voice message or standalone endpoint
- `README.md` + `metadata.json` at `integrations/voice-capture/`

### What to build
Endpoint that accepts audio, runs Whisper transcription (via API), and passes the text to `capture_thought`. Could extend the existing Telegram bot integration in `integrations/`.

---

## 4.3c: Browser extension

### Done when
- Chrome extension at `integrations/browser-capture/`
- Highlight text → right-click → "Capture to Open Brain"
- Popup for adding context before capture
- `README.md` + `metadata.json`

### What to build
Chrome extension with context menu integration. Popup UI for adding type/topic metadata before sending to `ingest-thought`.

## Dependencies

None — each is standalone. Slack and Discord integrations already exist in `integrations/`.
