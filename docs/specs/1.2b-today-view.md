# Feature 1.2b: Today view

**Category:** `dashboards/command-center` — build on the scaffold from 1.2a

## Done when

- Today view (`page.tsx`) shows four sections with live data from Supabase:
  1. **Due today / overdue** — actions with `due_date <= today` and `status = 'open'`. Red if overdue, amber if due today.
  2. **Upcoming this week** — actions due within 7 days + maintenance tasks due within 7 days + important dates within 7 days.
  3. **Unprocessed thoughts** — thoughts from last 48 hours of type `action_item` or `question` with no linked action. Each has a "Create action" button.
  4. **Stale loops** — actions open > 14 days with no recent related captures.
- StatsHeader shows live counts (open actions, overdue, thoughts this week, stale loops)
- ActionCard component has status controls: mark in_progress, complete (prompts for note), cancel
- Completing a recurring action shows the auto-spawned next instance immediately
- "Create action" button on unprocessed thoughts pre-fills content and links via `thought_id`

## What to build

### Components

**`ActionCard.tsx`** — displays an action with:
- Content, due date, tags, recurring badge if applicable
- Status controls: buttons for in_progress / complete / cancel
- Complete button opens a small input for `completion_note` (required)
- On status change, updates via Supabase client and refreshes the list

**`ThoughtCard.tsx`** — displays a thought with:
- Content, type badge, date
- "→ Action" button that creates a linked action (inserts into `actions` with `thought_id`)

**`UnprocessedQueue.tsx`** — queries thoughts where:
- `created_at > now() - interval '48 hours'`
- `metadata->>'type'` in (`action_item`, `question`)
- No matching row in `actions` where `actions.thought_id = thoughts.id`
- Renders each as a ThoughtCard with "Create action" affordance

**`StaleLoops.tsx`** — queries actions where:
- `status = 'open'` and `created_at < now() - interval '14 days'`
- Each card shows "Snooze 7 days", "Cancel", or "Do now" buttons
- Snooze = update `created_at` to now (resets the staleness clock)

**`StatsHeader.tsx`** — update from placeholder to live queries:
- Open actions: `select count(*) from actions where status = 'open'`
- Overdue: `select count(*) from actions where status = 'open' and due_date < today`
- Thoughts this week: `select count(*) from thoughts where created_at > now() - interval '7 days'`
- Stale loops: count of actions open > 14 days

### Data access

All queries use the Supabase JS client configured in 1.2a. Direct table queries — no RPC needed for this view except the unprocessed thoughts check (which is a left join or `not in` subquery).

### QuickActionBar

Wire up the submit handler from the 1.2a placeholder:
- "Capture thought" mode: POST to the `ingest-thought` Edge Function URL (from env var)
- "Create action" mode: insert directly into `actions` table via Supabase client
