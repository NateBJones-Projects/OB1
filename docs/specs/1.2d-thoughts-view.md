# Feature 1.2d: Thoughts view

**Category:** `dashboards/command-center` — build on scaffold from 1.2a

## Done when

- Thoughts view at `/thoughts` shows a reverse-chronological thought feed
- FilterBar component filters by type, topic, person, date range
- SearchBar component does semantic search via `match_thoughts` RPC
- Search results show a "Create action from this" button
- Every thought card has a "→ Action" button that creates a linked action
- Pagination or infinite scroll for large thought sets

## What to build

### Components

**`FilterBar.tsx`** — shared filter component:
- Type filter: dropdown of thought types from `metadata->>'type'`
- Topic filter: text input matching `metadata->>'topics'`
- Person filter: text input matching `metadata->>'people'`
- Date range: start/end date pickers
- Filters apply to the thought feed query

**`SearchBar.tsx`** — semantic search:
- Text input that calls `match_thoughts` RPC (pgvector similarity search)
- Results replace the feed temporarily
- Each result shows similarity score + "Create action from this" button

**Thought feed** — the main view:
- Reverse-chronological list of thoughts
- Uses ThoughtCard component from 1.2b
- Key interaction: select a thought → "Convert to action" creates an action linked via `thought_id`
- Paginate with `limit`/`offset` or cursor-based pagination

### Data access

- Feed: `select * from thoughts order by created_at desc limit 50 offset [page]`
- Filtered feed: add `where` clauses based on FilterBar state
- Search: call `match_thoughts` RPC with the search query text

## Dependencies

- 1.2a (scaffold) must be built
- 1.2b (today view) should be built first — provides ThoughtCard component
- 1.1a (UUID exposure) needed for thought → action linking
