# Feature 4.2: Weekly trend analysis (MCP tool)

**Category:** Add tool to `supabase/functions/open-brain-mcp/index.ts`

> This is the **on-demand MCP tool** mode. The scheduled mode (task type `trend_analysis`) is a separate spec at `docs/specs/1.1d-s8-trend-analysis.md`.

## Done when

- `weekly_trends` MCP tool exists in `open-brain-mcp/index.ts`
- Accepts `weeks_back` param (default 4)
- Computes and returns all five metrics listed below
- Output format matches the example below
- This test works:
  ```
  Show me weekly trends
  → Returns topic velocity, type shifts, people frequency, sentiment, capture cadence
  ```

## What to build

### MCP tool definition

| Tool | Params | Description |
|------|--------|-------------|
| `weekly_trends` | `weeks_back?` (default 4) | Compares this week's thought distribution against previous weeks. |

### Metrics to compute

1. **Topic velocity:** Topics from `metadata->>'topics'` — new this week (not in previous weeks), growing (increasing frequency), dropped (present in trailing average, absent this week).
2. **Type distribution shift:** % of thoughts by `metadata->>'type'` this week vs N-week average. Flag significant shifts (>25% change).
3. **People frequency:** Who from `metadata->>'people'` appeared more/less this week vs trailing average.
4. **Sentiment trend:** Average `metadata->>'sentiment'` score this week vs previous weeks.
5. **Capture cadence:** Thoughts per day this week vs average. Flag days with zero captures.

### Data queries

All data comes from the `thoughts` table. Query thoughts for the current week and for each of the `weeks_back` trailing weeks. Group and aggregate by the metadata fields.

### Output format

```
Weekly Trends (March 17-23 vs 4-week average)

GROWING: mold-remediation (5 captures, +3 vs avg), Fabriq (3, +2)
NEW THIS WEEK: Cody, consulting-transition
DROPPED: PDCS-2.0 (0 captures, avg was 2), MDM (0, avg 1.5)

TYPE SHIFT: action_items up 40% vs avg, questions down 25%
CAPTURE CADENCE: 4.1/day (avg 3.2) — no zero days this week
PEOPLE: Alexis (6 mentions, +3), Matt V (4, +2), Ingrid (0, was 2)
```

## Dependencies

None — standalone MCP tool. Quick win that immediately improves weekly review sessions.
