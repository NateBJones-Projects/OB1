Open Brain — Browser Bookmarklet (Qdrant Stack)

Capture selected text or the current URL to your Open Brain.

Capture to Open Brain (Qdrant)
  1. Drag the button above to your Chrome bookmarks bar.
  2. Navigate to any page. Select some text, or select nothing to capture the URL.
  3. Click the bookmark. On first use it will prompt for your OPEN_BRAIN_KEY and store it in localStorage.
  4. A confirmation toast appears for 3 seconds showing the captured type and topics.
Reset stored key

Run this in the browser console on any page:

localStorage.removeItem('ob_qdrant_key')
