# SPOV Generation

## Problem

How to transform accumulated insights into bold, actionable points of view that challenge conventional wisdom and guide decision-making.

## Trigger Conditions

- Generate SPOVs from insights
- Create truth/myth statements
- Transform knowledge into contrarian positions
- Apply confidence scoring to unconventional insights
- Combine insights from multiple domains

## Process

1. **Input Preparation**
   - Gather insights from DOK2 knowledge clusters and DOK3 strategic insights
   - Ensure each insight has clear source documentation
   - Identify potential connections between insights from different domains

2. **SPOV Generation**
   - Analyze patterns across insights
   - Develop contrarian positions that combine 2+ domains
   - Classify each SPOV as either "truth" or "myth"
   - Create provocative, actionable statements

3. **Confidence Scoring**
   - Assign confidence scores from 0.1 to 0.95
   - Use full range: 0.1-0.3 (speculative), 0.3-0.5 (plausible), 0.5-0.7 (moderately supported), 0.7-0.85 (well-supported), 0.85-0.95 (strongly supported)
   - Avoid clustering around 0.75-0.80

4. **Documentation**
   - Record supporting evidence (2-3 key facts)
   - Identify what conventional wisdom is challenged
   - Link to source insights that influenced the SPOV

## Output

A structured SPOV with:
- Title: Provocative name for the contrarian view
- Type: "truth" or "myth"
- Position: Bold statement of the contrarian perspective
- Evidence: 2-3 facts supporting the view
- Challenge: What conventional wisdom is challenged
- Confidence: Numeric score (0.1-0.95)
- Sources: Linked source insights

## Notes

- SPOVs go beyond simple observations to actionable decisions
- Truths are surprising things that ARE true but most people wouldn't expect
- Myths are widely believed things that are NOT actually true
- SPOVs should combine insights from multiple domains for maximum impact
- Maintain boldness while staying grounded in the evidence
